- Interaktiver Unterricht mit schrittweisen Anweisungen

- Android Wear Kurs

AIDE ist eine integrierte Entwicklungsumgebung (IDE) zum Entwickeln realer Android-Apps direkt auf Ihrem Android-Gerät. Folgen Sie interaktiven Codierungslektionen und werden Sie Schritt für Schritt zum erfahrenen App-Entwickler. Visuell gestaltete Apps schreiben Code mit dem funktionsreichen Editor mit Echtzeit-Fehlerüberprüfung, Refactoring und intelligenter Code-Navigation. Führen Sie Ihre App mit einem einzigen Klick aus und suchen Sie mit dem Java-Debugger nach Fehlern. AIDE bietet interaktive Lektionen mit schrittweisen Anweisungen zum Erlernen der Android-App-Entwicklung und der Java-Programmierkenntnisse. Folgen Sie den Lektionen, werden Sie Experte und wenden Sie das Gelernte in Ihren eigenen App-Projekten an. AIDE verwandelt Ihr Android-Tablet mit Tastatur in eine echte Entwicklungsbox. AIDE verwandelt Ihr Android Phone in einen kleinen Entwicklungscomputer, mit dem Sie Ihren Code unterwegs durchsuchen und berühren können. AIDE unterstützt das Erstellen von Apps mit JavaXml und die Android SDK-Apps mit CC und dem Android NDK sowie reine Java-Konsolenanwendungen. AIDE ist vollständig kompatibel mit Eclipse-Projekten. Sie können einfach den Quellcode auf Ihr Gerät kopieren und das Eclipse-Projekt in AIDE öffnen, um mit dem Codieren zu beginnen. Alternativ können Sie Ihren Quellcode auf Ihrer Dropbox behalten - AIDE lässt sich in Dropbox integrieren und ermöglicht das einfache Herunterladen von Ihrer Dropbox und das Synchronisieren Ihrer Änderungen. AIDE kann auch Android Studio-Projekte öffnen, die der Standardordnerstruktur folgen. AIDE unterstützt Git auch bei der beruflichen Weiterentwicklung. Beim Kauf von AIDE Prime the AIDE Premium Key deaktiviert der Debugger oder der UI-Designer alle Anzeigen in der App. - Integrierter LogCat Viewer Debugger (In-App-Kauf) - Debuggen von Android Java-Apps und Java Console-Apps - Haltepunkte - Step-In-Step-Out-Step-Out - Überprüfen der Thread-Ablaufverfolgung - Überprüfen lokaler Variablen - Überprüfen von Objektfeldern Echtzeit-Fehlerprüfung - Echtzeit-Fehleranalyse während des gesamten Projekts während der Eingabe - Automatische Schnellkorrekturen für viele Fehler UI-Design - Vorschau von XML-Layouts - Sprung von der Ansicht im Designer zum XML-Element Refactoring - Umbenennen
